# AI Prompts

Utility prompts for analyzing downloaded transcripts with LLMs.

| File | Purpose |
|------|---------|
| `summarize_transcripts_prompt.md` | Generate summaries for individual transcripts |
| `unified-theory.md` | Meta-analysis prompt for consolidated transcripts |

## Usage

1. Download and consolidate transcripts using the CLI tools
2. Copy the relevant prompt
3. Feed it to your preferred LLM along with the transcript content
