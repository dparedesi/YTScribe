The presentation begins by contextualizing Kubernetes adoption, with 80% of enterprises using Kubernetes in production according to CNCF 2024 survey data, driven by simplicity, consistency, and extensibility. ECR enhancements include enhanced scanning with Inspector integration providing live inventory of vulnerable running images, authenticated pull-through cache supporting ECR-to-ECR replication across regions and accounts, tag immutability flexibility for security and compliance, archival storage for compliance retention at lower costs, and managed image signing integrated with AWS Signer. EKS upgrade improvements feature cluster insights scanning for deprecated APIs and compatibility issues with on-demand refresh, Kubernetes version support acceleration delivering new versions within 45 days of upstream releases, and the EKS global dashboard providing centralized cross-account, cross-region inventory for executive-level visibility into cluster versions and upgrade status. Observability enhancements include enhanced container network observability with a single agent exposing metrics to CloudWatch, providing service maps showing pod-to-pod communication flows and traffic patterns, the hosted EKS MCP server integrated with Amazon Q for troubleshooting using built-in runbooks, and CloudWatch Container Insights with EBS metrics, detailed GPU metrics, and application signals support. Cost visibility improvements include integration with AWS Cost Allocation supporting Kubernetes labels and GPU cost attribution. EKS Capabilities represents a major expansion beyond the cluster, launching managed Argo CD with native Secrets Manager integration, CodeCommit credentials management, and managed networking sync traffic across accounts and regions, plus managed capabilities for ACK (AWS Controllers for Kubernetes) and Crossplane enabling Kubernetes-native infrastructure provisioning. Ultra Clusters enable unprecedented scale, supporting close to 100,000 nodes and up to 800,000 GPUs in a single cluster through three foundational innovations: moving BoltDB from network-attached storage to in-memory TEFS-based solution for order-of-magnitude performance improvements, partitioning key spaces allowing hard resource types to be split into separate ETCD clusters delivering up to 5x write throughput, and offloading consensus management from traditional Raft-based consensus to AWS's journal system enabling ETCD replica scaling without quorum requirements and multi-AZ durability. Additional performance improvements include multi-network interface support for pods enabling up to 100 gigabits per second bandwidth, concurrent download and unpacking of container images cutting pull times in half, prefix delegation for VPC address optimization improving node launch rates up to threefold, and auto-repair capabilities for all compute including accelerated instances. Provisioned Control Plane brings ultra-scale performance benefits to all clusters through pre-allocated capacity tiers, eliminating dynamic scaling uncertainty and providing consistent predictable performance with capabilities like processing up to 6,800 concurrent API requests in the highest tier. Netflix's Neil Mullen shares their migration journey, describing how they migrated their entire Titus container platform fleet to EKS within one quarter despite operating at massive scale with hundreds of thousands of containers across four regions, handling launch rates of 70,000 containers in 5-minute periods during region evacuations, and successfully moving from self-managed control planes to EKS while maintaining their high-scale multi-tenant architecture with clusters supporting up to 80,000 pods. Future priorities include supporting critical workload patterns at any scale potentially requiring multi-cluster solutions, expanding AWS service integrations as Kubernetes becomes the front door to cloud for many customers, meeting workloads where they are from edge to cloud, simplifying platform building through more managed capabilities, and accelerating innovation through continued open source community engagement.
