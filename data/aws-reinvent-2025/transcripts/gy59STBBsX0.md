He shows how to auto-instrument with ADOT, enable transaction search, and gain unified visibility into agent performance, sessions, traces, and error metrics. The demo highlights drilling down into trajectory maps to pinpoint failures (e.g., nutrition service failing to find pet type) and viewing chat spans with masked PII. Jeff also illustrates CloudWatch APM for troubleshooting the entire application stack, tracing failures from front-end to payment processing, and leveraging automatically generated operational audits. CloudWatch MCP servers bring observability to developers' IDEs (like Kira), allowing natural language troubleshooting. Application Observability for AWS GitHub Action integrates production telemetry directly into GitHub workflows, enabling AI to diagnose issues and auto-generate pull requests with fixes.
