The core attack vectors are categorized into four main types: prompt injection (manipulating LLM instructions through user input, URLs, PDFs, or emails), tool poisoning (manipulating API calls or function calls with harmful parameters to execute unauthorized actions), agent-to-agent escalation (hijacking multi-step chains where one agent calls another with escalated privileges), and supply chain corruption (poisoning knowledge bases, metadata, or databases through malicious document uploads that get indexed and pollute retrieval results).
