  AWS presenters Kuram, Nitz, and Vijay showcase AWS Transform, a free agentic AI platform that modernizes Windows stacks end-to-end by converting .NET Framework apps and SQL Server databases to .NET 8/10, modern UI frameworks (Blazor/MVC), and PostgreSQL (Aurora/Postgres) running on Linux EC2/ECS.
